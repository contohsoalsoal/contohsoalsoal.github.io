39. tolong bantu matematika peminatan kelas xi


Mapel : Matematika
Bab : Trigonometri

Buktikan !
[tex] \frac{ \cos x . \tan^{2}x }{ \sin x } = \tan x[/tex]
Informasi =
tan x = sin x / cos x
Jadi,
cos x / sin x = cot x atau 1/tan x
[tex]{ \frac{1}{ \tan x} . \tan^{2}x } = \tan x \\ \tan x = \tan x[/tex]
Jadi, cos x . tan^2 x / sin x = tan x
