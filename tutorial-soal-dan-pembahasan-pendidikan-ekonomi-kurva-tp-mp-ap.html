11. Diketahui fungsi produksi : TP = Y = 27 X2 – 2 X3, X adalah penggunaan input Tentukan : a) TP maksimum b) MP maksimum c) AP pada saat X = 4


Jawaban:

Untuk menentukan TP maksimum, kita perlu mencari titik di mana turunan kedua fungsi produksi terhadap input (X) sama dengan nol. Dalam hal ini, kita akan mencari titik maksimum dengan memeriksa perubahan tanda dari turunan kedua.

a) TP Maksimum:

Untuk mencari titik maksimum TP, kita harus mencari turunan pertama dan kedua fungsi produksi terhadap X.

TP = 27X^2 - 2X^3

Turunan pertama terhadap X:

TP' = 54X - 6X^2

Turunan kedua terhadap X:

TP'' = 54 - 12X

Untuk menentukan TP maksimum, kita atur TP'' = 0:

54 - 12X = 0

12X = 54

X = 4.5

Menggunakan uji perubahan tanda, kita dapat menentukan bahwa TP mencapai maksimum pada X = 4.5.

b) MP Maksimum:

Untuk menentukan MP maksimum, kita perlu mencari turunan pertama fungsi produksi terhadap X.

TP' = 54X - 6X^2

MP adalah turunan pertama fungsi produksi terhadap X, jadi:

MP = 54X - 6X^2

Untuk menentukan MP maksimum, kita dapat mencari turunan kedua MP terhadap X:

MP' = 54 - 12X

Kita set MP' = 0 untuk mencari nilai X yang menghasilkan MP maksimum:

54 - 12X = 0

12X = 54

X = 4.5

Menggunakan uji perubahan tanda, kita dapat menentukan bahwa MP mencapai maksimum pada X = 4.5.

c) AP pada saat X = 4:

AP (Average Product) adalah TP dibagi oleh jumlah input yang digunakan (X):

AP = TP / X

Untuk mencari AP saat X = 4, kita substitusikan nilai X = 4 ke dalam fungsi produksi:

TP = 27X^2 - 2X^3

TP = 27(4)^2 - 2(4)^3

TP = 27(16) - 2(64)

TP = 432 - 128

TP = 304

Kemudian, kita bagi TP dengan X:

AP = TP / X

AP = 304 / 4

AP = 76

Jadi, AP pada saat X = 4 adalah 76.

23. Hubungan TP, MP, dan AP dalam The Law of Diminishing Return


TP : TOTAL PRODUCT

MP : MARGINAL PRODUCT

AP : average product

hubungannya dalam deminishing of return adalah dalam tahapantahapannya dalam hal produksi yang dapat dibagi ke dalam tiga tahap, yaitu :
a. produksi total dengan increasing returns,
b. produksi total dengan decreasing returns, dan
c. produksi total yang semakin menurun.


1. Tahap Pertama : Produksi Total (Total Product)mengalami pertambahan yang semakin cepat.

Tahap ini dimulai dari titik origin semakin kesatu titik pada kurva total product dimana AP (produksi rata-rata) maksimum, dan pada titik ini AP=MP (marginal product).

2. tahap Kedua : Produksi Total (Total Product) pertambahannya semakin lama semakin kecil.
Tahap II ini dimulai dari titik APTmaksimum sampai titik dimana MP=0, atau TP maksimum.

3. Tahap Ketiga : Produksi total (total product) semakin lama semakin menurun.

Tahap III ini meliputi daerah dimana MP negatif
