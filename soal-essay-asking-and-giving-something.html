12. tolong buatkan expression of asking for and giving something



1. To ask for things, we can use the following sentences:
- Can I…., please?
- May I…., please?
- Can you…., please?

2. To give the things, we can use the following sentences:
- Here you are.
- Here it is.
- Okay
- Sure.
- Certainly
- Of Course.
- I would be glad/happy to ........

3. To refuse to give the things, we can use the following sentences:
- I’m sorry. I’m using it.
- Not now. I’m using it. Sorry.
- Not now. I’m using them. Sorry
- Sorry, I’m using it right now.
- Sorry, I’m using them right now.

4. To show our gratitude, we can use the following sentences:
- Thanks.
- Thanks a lot
- Thank you
- Thank you very much

5. To answer the gratitude expression, we can use the following sentences:
- You're welcome
- It's OK
- My pleasure
- No problem
- Don't mention it
