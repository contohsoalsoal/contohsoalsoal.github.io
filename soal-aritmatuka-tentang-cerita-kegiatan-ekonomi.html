4. tentukan jumlah dari deret aritmatuka berikut ini ?A.47+49+51+.....+101B.72+70+68+.....+40


A. 
a = 47
b = U2 - U1 = 2

Un = a + (n-1) b
101 = 47 + (n-1) 2
101 = 47 + 2n - 2
101 - 45 = 2n
56 / 2 = n
28 = n

Sn  = n/2 (a + Un)
S28 = 28/2 (47 + 101)
S28 = 14 (148)
S28 = 2072

B.

40 = 72 + (n-1) -2

40 - 74 = -2n

17 = n

Sn = n/2 (a + Un)

11. Suatu barisan aritmatuka memili aU6=38dan U9=59.Jumlah sepuluh suku pertama barisan itu adalah


Baris dan Deret Aritmatika

Rumus
Un = a + (n - 1)b
Sn = n/2 (2a + (n - 1)b)

Soal
U6 = 38
U9 = 59
S10 = ...

Penyelesaian

Menentukan nilai b (beda)
U9 = U6 + 3b
59 = 38 + 3b
3b = 59 - 38
3b = 21
b = 21 / 3
b = 7

Menentukan Nilai a (suku pertama / U1)
U6 = a + 5b
38 = a + 5(7)
38 = a + 35
a = 38 - 35
a = 3

Menentukan Jumlah Suku ke 10
S10 = 10 / 2 (2(3) + (10 - 1)7)
➡ 5 (6 + (9)7)
➡ 5 (6 + 63)
➡ 5 (69)
➡ 345
